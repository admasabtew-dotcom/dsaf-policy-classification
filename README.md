# Dynamic Sparse Attention Fusion (DSAF)

## Overview
This repository contains the implementation of a hybrid ontology–deep learning model for multilingual policy classification.

## Dataset
The dataset includes:
- Amharic, English, Oromo, and bilingual policy texts
- 15 policy domain classes
- Train/validation/test splits

## Structure
- data/: dataset and statistics
- scripts/: preprocessing scripts
- figures/: dataset visualizations
